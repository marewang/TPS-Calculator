# Kalkulator Kapasitas (DAU • Concurrency • TPS)

Vite + React + Tailwind. Siap deploy ke **Vercel**.

## Jalankan Lokal
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy ke Vercel
- Framework preset: **Vite**
- Build command: `npm run build`
- Output dir: `dist`
- Node: 18+
